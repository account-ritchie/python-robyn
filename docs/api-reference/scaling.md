## 扩展应用程序

蝙蝠侠将他的应用程序扩展到多个核心以获得更好的性能。他使用了以下命令：

::: code-group

```py [untyped]
python3 app.py --workers=N --process=M
```

```py [typed]
python3 app.py --workers=N --process=M
```

:::
