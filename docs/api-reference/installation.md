## 安装 Robyn

Robyn 是一个 Python 库，你可以使用 `pip` 或 `conda` 安装

```sh
pip install robyn
```

虽然 Robyn 还有其他扩展，例如

```sh
pip install "robyn[templating]"
```

建议先安装基础包，然后根据需要安装扩展包。
